## ATS Project Description

*Optimized for Applicant Tracking Systems — includes key technical terms.*

### 3-Line Version

> EduRisk AI is an end-to-end machine learning system for predicting university student academic risk levels. Built with Python, scikit-learn, XGBoost, SHAP, FastAPI, Next.js, and Docker. Achieves 85.58% accuracy and 94.92% ROC-AUC on 27,901 student records with per-prediction SHAP explainability.

### Full Version

> EduRisk AI — a production-oriented ML platform for student academic risk prediction. Engineered a leakage-free preprocessing pipeline (imputation, label encoding, StandardScaler) and composite risk scoring for 3-class target engineering. Trained and compared 4 classifiers (Random Forest, XGBoost, SVM, MLP) with GridSearchCV and Optuna Bayesian hyperparameter tuning. Best model: Random Forest (85.58% accuracy, 94.92% ROC-AUC, 85.27% 3-fold CV). Implemented SHAP TreeExplainer for per-prediction explainability with human-readable risk factor interpretations. Built FastAPI REST API with Pydantic validation and auto-generated Swagger documentation. Developed Next.js dark-mode frontend with animated risk gauges, SHAP waterfall visualizations, and personalized recommendations. Docker containerized with CI/CD pipeline (GitHub Actions, pytest, flake8, black). 62 unit tests across all modules.

## Technical Highlights

### Architecture

| Component | Technology | Why |
|-----------|-----------|-----|
| ML Pipeline | Python, scikit-learn, XGBoost | Industry standard for tabular data |
| Explainability | SHAP TreeExplainer | Mathematical guarantees, per-prediction |
| API | FastAPI + Pydantic | Auto-docs, type safety, async |
| Frontend | Next.js + TypeScript + Tailwind | Production UI, dark mode, animations |
| Testing | Pytest (62 tests) | All modules covered |
| DevOps | Docker, GitHub Actions | Reproducible, automated |
| Config | Python dataclasses | Type-safe, IDE-friendly |

### Key Metrics

| Metric | Value |
|--------|-------|
| Dataset | 27,901 students, 11 features |
| Best Accuracy | 85.58% (Random Forest) |
| Best ROC-AUC | 94.92% (Random Forest) |
| Cross-validation | 85.27 ± 0.39% (3-fold) |
| Unit Tests | 62 passing |
| API Endpoints | 3 (/predict, /health, /model/info) |
| Frontend Components | 6 custom React components |

### What Makes This Project Stand Out

1. **Full lifecycle** — not just a notebook, but data → training → API → frontend → tests → deployment
2. **Explainability** — SHAP per-prediction explanations, not just feature importance
3. **Engineering quality** — 62 tests, no data leakage, centralized config
4. **Multiple interfaces** — REST API + Next.js + Gradio
5. **Professional documentation** — case study, interview prep, brand guide
